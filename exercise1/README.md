## Task
Given an integer array, nums, move all of the zeros to the end of it while maintaing the relative order of non-zero elements.


## How to run?

Run example
```
go run main.go
```

This should be expected result
```
go run main.go
[0 1 2 3 40 0 2 3 0]
[1 2 3 40 2 3 0 0 0]
```


## Tests

Run tests
```
go test
```
